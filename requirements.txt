# Crypto Market Data Pipeline

A modular ETL pipeline built using Python and Pandas that fetches cryptocurrency market data from the Zebpay API, transforms and cleans the data, and stores it in Parquet format for analytics.

---

## Project Architecture

```text
API → Ingestion → Transformation → Load → Parquet Storage
```

---

## Features

- API-based data ingestion
- JSON to DataFrame conversion
- Data cleaning and transformation
- Null handling and datatype validation
- Parquet file generation
- Modular ETL architecture

---

## Tech Stack

- Python
- Pandas
- Requests
- PyArrow

## How to Run

### Clone repository

```bash
git clone <repo-url>
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run pipeline

```bash
python pipeline.py
```
